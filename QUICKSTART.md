# ⚡ 5分钟快速上手指南

这是一个精简的快速开始指南，让你在5分钟内掌握最常用的功能。

## 📦 安装（1分钟）

```bash
git clone https://github.com/YOUR_USERNAME/vim-ide-enhanced.git
cd vim-ide-enhanced
bash install.sh
```

等待插件安装完成。

## 🎮 第一次使用（2分钟）

### 1. 打开文件

```bash
cd /path/to/your/project
vim your_code.c
```

### 2. 记住这3个键

| 按键 | 功能 | 
|------|------|
| `F3` | 打开文件树 |
| `F4` | 打开函数列表 |
| `Ctrl-p` | 搜索文件 |

### 3. 试一试

1. 按 **F3** → 看到左边的文件树
   - 用 `↑↓` 选择文件
   - 按 `Enter` 打开
   - 按 `q` 关闭

2. 按 **F4** → 看到右边的函数列表
   - 用 `↑↓` 选择函数
   - 按 `Enter` 跳转
   - 按 `q` 关闭

3. 按 **Ctrl-p** → 搜索框出现
   - 输入文件名（模糊匹配）
   - 按 `Enter` 打开
   - 按 `Esc` 取消

## ✏️ 基本编辑（1分钟）

### 保存和退出

```
,w     保存文件（逗号+w）
,q     退出vim（逗号+q）
,x     保存并退出（逗号+x）
```

### 代码折叠

```
空格键   折叠/展开当前代码块
```

### 快速注释

```
gcc    注释/取消注释当前行
```

在 Visual 模式选中多行后：
```
gc     注释/取消注释选中的行
```

## 🪟 窗口切换（1分钟）

```
Ctrl-h   跳到左边窗口
Ctrl-j   跳到下方窗口
Ctrl-k   跳到上方窗口
Ctrl-l   跳到右边窗口
```

## 🎯 实战练习

### 练习1: 浏览项目

1. `vim` （不带参数打开）
2. 按 `F3` 打开文件树
3. 浏览目录，找到一个 `.c` 文件
4. 按 `Enter` 打开
5. 按 `F4` 查看函数列表
6. 选择一个函数按 `Enter` 跳转

✅ 完成！你已经会使用文件浏览器了。

### 练习2: 快速搜索

1. 在项目目录打开 vim
2. 按 `Ctrl-p`
3. 输入 `main` （或其他文件名）
4. 看到匹配的文件列表
5. 按 `Enter` 打开

✅ 完成！你已经会快速搜索文件了。

### 练习3: 代码编辑

1. 打开一个代码文件
2. 移动光标到一个函数开始处
3. 按 `空格键` 折叠函数
4. 再按 `空格键` 展开
5. 移动到一行代码
6. 按 `gcc` 注释这行
7. 再按 `gcc` 取消注释
8. 按 `,w` 保存

✅ 完成！你已经会基本编辑操作了。

### 练习4: 多窗口

1. 在 vim 中输入 `,vs` 垂直分屏
2. 按 `Ctrl-p` 打开另一个文件
3. 按 `Ctrl-h` 和 `Ctrl-l` 在两个窗口间切换
4. 按 `F3` 打开文件树
5. 现在你有三个窗口了！
6. 用 `Ctrl-h/j/k/l` 随意切换

✅ 完成！你已经会多窗口操作了。

## 🆘 遇到问题？

### Q: F4 不工作？

**A:** 需要先生成 tags：
```bash
cd /path/to/project
ctags -R .
```

### Q: 想要退出 vim？

**A:** 
```
:q      退出
:q!     强制退出不保存
,q      快捷键退出
```

### Q: 如何撤销操作？

**A:**
```
u       撤销
Ctrl-r  重做
```

### Q: 忘记快捷键了？

**A:** 打开 [CHEATSHEET.md](CHEATSHEET.md) 查看，或者打印出来放在桌边。
