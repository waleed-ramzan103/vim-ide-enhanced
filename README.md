## üé® Themes and Appearance

Choose from 9 beautifully crafted themes to customize the appearance of your IDE. You can switch themes easily by changing the settings in your `.vimrc` file. 

### To Change Theme:
1. Open `.vimrc` again:
   ```bash
   nano ~/.vimrc
   ```
2. Locate the theme setting, usually at the top. It should look like:
   ```vim
   colorscheme your-chosen-theme
   ```
3. Replace `your-chosen-theme` with the name of the theme you like.
